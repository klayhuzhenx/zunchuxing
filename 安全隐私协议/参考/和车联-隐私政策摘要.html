一、个人信息收集规则

根据《常见类型移动互联网应用程序必要个人信息范围规定》，和车联核心归类为地图导航类，兼具用车服务类复合属性，实现基础业务功能的必要个人信息包括：

  • 用户电话号码；
  • 用户以及车辆位置信息、行踪轨迹；
  • 车辆信息、车队信息（租赁 / 物流场景）。

各业务功能收集的信息范围如下：

业务功能模块 核心收集信息 拒绝提供的影响
账号登录与个人资料管理 手机号、微信 OpenID（微信登录）、用户名、头像、登录 IP、登录 / 操作日志信息 无法完成账号登录，无法使用平台基础服务
车联网核心服务（车辆数据看板、实时定位、历史轨迹、车辆列表、车队管理） 车辆唯一标识信息（底盘号、车牌号等）、车辆实时 / 历史位置信息、运行状态信息、车队基础信息、人员管理信息（姓名、手机号、身份证号用于身份核验）、账号权限信息 无法使用车辆定位、轨迹回放、车队管理等核心车联网服务
统计分析服务（运行分析、充电分析、驾驶行为统计、能耗报表） 车辆行驶 / 充电 / 能耗 / 驾驶行为相关数据、位置信息、司机身份信息、车队归属信息 无法查看对应运营数据统计与分析内容
安全与适配保障 设备型号、操作系统版本、网络信息、位置相关信息、访问日志信息 无法保障 App 正常适配运行、账号安全与风险防控

法定豁免情形：为履行法定义务、应对突发公共安全事件、处理合法公开信息等法律法规规定的情形，我们收集 / 使用您的个人信息无需征得您的授权同意。

设备权限 核心调用目的 对应业务功能
位置权限 获取车辆定位坐标、加载地图数据、标记充电 / 异常驾驶位置 车辆实时定位、历史轨迹、寻车、充电分析、驾驶行为统计

存储权限 缓存轨迹 / 报表数据、存储个人设置与政策文档、提升加载体验 历史轨迹、统计分析服务、个人信息与设置服务

三、第三方 SDK 使用说明

我们仅为实现特定功能接入第三方 SDK，并对 SDK 进行严格安全检测；SDK 相关权限调用会单独征求您的同意。接入详情如下：

  • 腾讯地图定位 SDK：用于实现用户定位、地图展示、车辆定位 / 轨迹展示、搜索导航服务，收集位置相关信息。

四、个人信息使用规则

  • 我们仅将您的个人信息用于本政策列明的、与实现服务目的直接或合理关联的场景，不会超出范围使用。
  • 我们会对去标识化的个人信息进行统计分析，用于产品优化、新功能开发，统计信息不会包含您的身份识别信息。
  • 我们仅会向您发送服务必需的验证码、服务提醒短信，绝不会通过短信、电话等方式开展任何营销、推销活动。
  • 超出原定目的使用您的个人信息，我们会再次征得您的明示同意。

五、个人信息的共享、转让与公开披露

  • 共享：我们不会擅自将您的个人信息与第三方共享，仅在获得您的明确同意后，向您指定的第三方共享授权范围内的信息。
  • 转让：如发生收购、兼并、重组等主体变更，我们会要求变更后的主体继续遵守本隐私政策；如需变更个人信息使用目的，会事先征得您的明示同意。
  • 公开披露：除非事先获得您的明确同意、依照法律规定必须披露，或为保护相关方合法权益且对信息去标识化处理，我们不会公开披露您的个人信息。

六、个人信息的存储与安全保护

  • 存储地域：我们收集的您的个人信息，全部在中国内地存储和使用；如需向境外传输，会另行告知并征得您的同意。
  • 保存期限：您使用服务期间，我们会持续保存您的个人信息；您注销账号后，我们将在法律法规规定的最短必需期限届满后，对您的个人信息进行删除或匿名化处理；违法违约记录、涉刑事案件相关信息将按法规要求延长保存期限。
  • 安全保障：我们建立了数据分类分级、全生命周期安全管控体系，对个人敏感信息加密存储，采用加密传输、严格权限控制、安全审计等技术与管理措施保障信息安全；已通过信息系统安全等级保护三级备案、ITSS 三级、CMMI 三级认证。
  • 安全事件处置：发生个人信息安全事件后，我们将按法规要求及时告知您事件详情、处置措施、风险防范建议，并向监管部门上报处置情况。

七、您的个人信息主体权利

您对您的个人信息享有以下法定权利，我们将在验证您的身份后15 日内对您的请求作出答复或合理解释：

  • 查询、复制、修改、删除权：您可通过【我的】-【设置】查询、复制、修改您的个人信息；可通过联系官方客服提交个人信息删除申请，注销账号可同步申请删除全部个人信息。
  • 撤回同意权：您可通过【我的】-【隐私管理】调整授权范围、撤回隐私政策授权，也可通过设备 / 小程序设置关闭相关权限；撤回授权不影响此前基于您的同意开展的信息处理行为。
  • 账号注销权：您可通过联系和车联人工客服提交账号注销申请，注销后我们将停止为您提供服务，并按法规要求处理您的个人信息。
  • 异议与投诉权：您可通过本政策列明的渠道反馈您的意见、建议或投诉。

响应例外：对于与履行法定义务、国家安全、公共安全、刑事侦查、商业秘密等法定情形相关的请求，我们将无法响应。

八、政策更新与平台关停

  • 政策更新：我们会根据业务迭代、法规要求修订本政策，更新后将通过平台 “我的 - 设置 - 隐私政策” 发布最新版本，并通过首页公告提示；未经您的明确同意，我们不会削减您按本政策享有的权利。
