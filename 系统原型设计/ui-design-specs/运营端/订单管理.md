# 尊出行 · 运营端订单管理模块 UI 设计说明

> PC Web 后台。MVP 阶段含包车出行和租车出行两类订单。

---

## 页面清单

> 本模块包含 **两个一级 Tab 视图**：订单、司机订单。每个 Tab 下含列表 + 详情侧抽屉 + 派车弹窗。

### 4.1 Tab 一：订单

#### 一级 Tab 切换
- 订单 | 司机订单

#### 二级状态 Tab

| Tab | 包含 |
|---|---|
| 全部 | 所有订单（默认） |
| 待支付 | 已下单未支付 + 待补款（列表内子状态标识） |
| 待派车 | 已支付未派车 |
| 待开始 | 已派车未到出发时间 |
| 行程中 | 已开始未结束 |
| 已完成 | 行程结束、款项结清 |
| 已取消 | 乘客取消（含未派车/已派车/提前结束） |

#### 筛选区

| 筛选 | 类型 |
|---|---|
| 订单类型 | 包车出行 / 租车出行（多选） |
| 支付方式 | 个人支付 / 企业支付（多选） |
| 关键字 | 订单号 / 乘客姓名+手机号 / 企业名称 / 司机姓名 |
| 出发时间 | 日期范围 |
| 下单时间 | 日期范围 |
| 企业 | 下拉+搜索（仅企业支付） |
| 司机 | 下拉+搜索（已派车） |
| 子状态 | 待支付 / 待补款（待支付 Tab 下） |

#### 列表列

| 列 | 内容 |
|---|---|
| 订单号 | 点击进详情 |
| 类型 | 包车/租车标签 |
| 下单人 | 姓名 + 手机号 |
| 支付方 | 个人 / 企业（企业名） |
| 用车时间 | 起止时间 |
| 上车地点 | 简略地址 |
| 司机 | 姓名（未派车红字"待派车"） |
| 车辆 | 车牌+车型 |
| 订单金额 | 实付金额 |
| 状态 | 状态标签 |
| 下单时间 | YYYY-MM-DD HH:mm |
| 操作 | 详情 / 派车 / 改派 |

#### 列表操作

| 操作 | 适用状态 | 行为 |
|---|---|---|
| 详情 | 全部 | 侧抽屉打开 |
| 派车 | 待派车 | 派车弹窗 |
| 改派 | 待开始/行程中 | 派车弹窗 |
| 导出 | 全部 | 当前筛选结果导出 Excel |

### 4.3 派车弹窗

#### 触发条件

| 状态 | 按钮 | 行为 |
|---|---|---|
| 待派车 | 派车 | 首次分配车辆/司机 |
| 待开始/行程中 | 改派 | 重新分配；中止当前司机订单生成新订单 |

#### 弹窗内容（左右两栏）

**左栏：订单概览**
- 订单号、类型、用车时段、上车点、下车点、乘客信息

**右栏：车辆与司机**

车辆筛选：
- 车型下拉（按订单预填可改）
- 车牌号输入/选择
- 仅展示"在用"状态车辆

工作日历：
- 月历视图，默认当月，可切换月份
- 每天格子内展示派车任务条（蓝：包车 / 橙：租车 / 灰：已结束 / 红：冲突时段）
- 当前订单以虚线框高亮
- 点击任务条 Tooltip 展示详情，可跳转订单
- 冲突时段红色提示，但不强制阻断

司机选择：
- 默认带出车辆绑定司机，可下拉更换
- 仅展示"在岗"司机
- 冲突仅提示不阻断

派车备注：选填 ≤200 字

#### 提交后

- 生成/更新司机订单，订单状态从待派车流转到待开始
- 司机端推送派车消息
- 乘客端推送"已派车"通知（车牌、司机姓名/电话）
- 改派：原司机订单作废，原司机收到取消通知，新司机收到派车通知

### 4.4 订单详情（侧抽屉）

#### 打开方式
侧抽屉，宽度约 60% 屏幕。

#### 区块内容

| 区块 | 内容 |
|---|---|
| 基本信息 | 订单号、类型、状态、子状态（待补款）、下单时间 |
| 下单人信息 | 姓名、手机号、企业、用户身份 |
| 用车信息 | 用车时间、上车点、下车点、用车要求 |
| 司机与车辆 | 司机+车辆；未派车红字"待派车" |
| 关联司机订单 | 列表，点击跳转司机订单详情 |
| 费用明细 | 基础费、超时费、超里程费、实付、退款 |
| 支付与结算 | 支付方式、时间、流水号、补款/退款记录 |
| 时间节点 | 下单→支付→派车→出车→开始→结束 |
| 备注 | 乘客备注（只读）+ 内部备注（运营可写） |
| 操作日志 | 谁什么时间做了什么 |

#### 详情页操作

| 操作 | 适用状态 |
|---|---|
| 派车/改派 | 待派车/待开始/行程中 |
| 添加内部备注 | 全部 |
| 触发退款 | 已取消/已完成 |
| 触发补款链接 | 待补款 |
| 导出 PDF | 已完成 |

> 运营端不可手动修改订单状态、取消订单、修改用车信息。

## 订单状态流转

| 当前状态 | 触发事件 | 流转到 |
|---|---|---|
| 待支付 | 乘客完成支付 | 待派车 |
| 待支付 | 超时未支付 30 分钟 | 已取消 |
| 待派车 | 运营派车 | 待开始 |
| 待派车 | 乘客取消 | 已取消（全额退） |
| 待开始 | 司机开始行程 | 行程中 |
| 待开始 | 乘客取消 | 已取消（按规则退） |
| 行程中 | 司机结束无超时超里程 | 已完成 |
| 行程中 | 结束产生超时超里程 | 待补款 |
| 行程中 | 乘客提前结束（多日租车） | 已完成（已开始不退、未开始按规则退） |
| 待补款 | 乘客完成补款 | 已完成 |
| 待补款 | 7 天未补款 | 财务催收（保持待补款，标红） |

## 页面跳转关系

```
工作台 → 订单管理（Tab：订单 | 司机订单）
            ├──→ 订单详情（侧抽屉）
            │      ├── 派车弹窗
            │      ├── 添加备注
            │      └── 导出 PDF
            ├──→ 司机订单详情（侧抽屉）
            └──→ 派车弹窗（列表/详情触发）
```
