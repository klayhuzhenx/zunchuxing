# 尊出行 · 运营端企业客户管理模块 UI 设计说明

> PC Web 后台。企业来源：小程序申请入驻 + 运营后台直接添加。

---

## 页面清单

> 本模块共 **3 张页面**：企业列表、企业详情、新增企业弹窗。

### 3.1 企业列表

#### 页面内容

- 标题栏：标题"企业客户管理" + 右上角"新增企业"按钮
- 筛选区：
  - 企业状态：待审核 / 已通过 / 已驳回 / 已禁用（多选）
  - 创建来源：小程序申请 / 后台添加（多选）
  - 关键字搜索：企业名称 / 统一社会信用代码 / 联系人手机号
  - 注册时间：日期范围
  - 剩余额度：大于/小于/等于 + 金额输入
- 列表列：企业名称（点击进详情）、企业编号、统一社会信用代码、联系人姓名+手机号、员工数、总额度、已使用、剩余额度、状态标签、创建来源、注册时间、操作
- 底部：批量导出 Excel

#### 状态标签

| 状态 | 标签色 |
|---|---|
| 待审核 | 琥珀色 |
| 已通过 | 绿色 |
| 已驳回 | 灰色 |
| 已禁用 | 红色 |

#### 列表操作

| 操作 | 适用状态 | 行为 |
|---|---|---|
| 详情 | 全部 | 跳转企业详情页 |
| 审核 | 待审核 | 进入详情页审核 |
| 调整额度 | 已通过 | 弹出调整额度弹窗 |
| 启用/禁用 | 已通过/已禁用 | 弹窗填写原因确认 |

#### 交互行为

- 点击"新增企业" → 弹出新增企业弹窗
- 点击行 → 跳转企业详情
- 支持按注册时间/已使用/剩余额度排序
- 批量导出当前筛选结果为 Excel

---

### 3.2 新增企业弹窗（运营手动添加）

#### 弹窗内容

- 弹窗标题："新增企业"
- 企业名称输入框（≤50 字）
- 统一社会信用代码输入框（18 位）
- 联系人姓名输入框（≤20 字）
- 联系人手机号输入框（11 位，不可重复）
- 备注输入框（选填，≤200 字）
- "确认创建"按钮

#### 交互行为

- 提交后直接置为"已通过"，无需审核
- 以联系人手机号为初始账号自动创建企业管理员
- 初始额度为 0，需通过"调整额度"开通
- 短信通知联系人：企业账户已创建

---

### 3.3 企业详情

#### 页面内容

**基本信息**
- 企业名称、统一社会信用代码、联系人姓名、联系人手机号
- 营业执照图片预览（小程序申请时上传，运营新增可后续补传）

**管理员信息**
- 管理员姓名、手机号、状态

**额度概览**
- 总额度、已使用、剩余额度、最近调整时间

**员工概览**
- 当前员工数（点击展开列表，仅展示不可操作）

**消费记录**（表格，默认近 30 天）
- 列：时间、类型（消费红色-/退款绿色+）、金额、关联订单号（点击跳转）、操作员工、出行场景
- 支持按月/日期范围筛选、按员工筛选、导出 Excel

**额度变动记录**
- 历次调整：调增/调减金额、操作员、时间、备注

**状态记录**
- 启用/禁用历史，含操作员和原因

#### 详情页操作（按状态）

| 状态 | 可操作 |
|---|---|
| 待审核 | 通过 / 驳回 / 编辑基本信息 |
| 已通过 | 编辑基本信息 / 调整额度 / 禁用 |
| 已驳回 | 编辑基本信息 / 重新审核 |
| 已禁用 | 启用（恢复历史数据） |

#### 审核操作

| 操作 | 行为 |
|---|---|
| 点击"通过" | 弹窗确认 → 状态变更为已通过 → 申请人成为管理员 |
| 点击"驳回" | 填写驳回原因（必填 ≤200 字） → 状态变更为已驳回 |

#### 调整额度弹窗

| 字段 | 说明 |
|---|---|
| 调整类型 | 调增 / 调减（单选） |
| 调整金额 | 必填正整数，调减不可使剩余额度低于 0 |
| 调整原因 | 必填 ≤200 字 |
| 操作员 | 自动记录当前登录账号 |

- 提交后立即生效，写入额度变动记录
- 调增 ≥ ¥10,000 时短信通知企业管理员

## 页面跳转关系

```
工作台 → 企业列表
            ├──→ 企业详情
            │      ├── 通过/驳回（审核弹窗）
            │      ├── 调整额度（弹窗）
            │      └── 禁用/启用（弹窗）
            ├──→ 新增企业（弹窗）
            └──→ 订单详情（从消费记录跳转）
```
