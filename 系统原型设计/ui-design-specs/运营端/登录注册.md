# 尊出行 · 运营端登录注册模块 UI 设计说明

> PC Web 后台，不开放注册。所有账号由超级管理员在系统管理中创建。

---

## 页面清单

> 本模块共 **2 张页面**：登录页、忘记密码页。

### 1.1 登录页

#### 页面内容

- 品牌标识：品牌徽标 + "尊出行运营管理平台"
- 极验行为验证（滑块/点选），每次登录均需通过
- 账号输入框（4-20 位字母/数字/下划线）
- 密码输入框（支持明文/密文切换）
- "登录"按钮
- 底部入口："忘记密码？"

#### 交互行为

- 点击"登录" → 极验通过 + 账号密码校验 → 成功跳转工作台
- 账号/密码错误/账号被禁用 → 统一提示"账号或密码错误"
- 账号被锁定 → 提示"账号已锁定，请联系超级管理员"
- 连续失败 10 次 → 账号锁定
- 点击"忘记密码" → 跳转忘记密码页
- 登录态 8 小时，连续 30 分钟无操作自动失效
- 同一账号仅允许一处登录，新登录踢掉旧会话

---

### 1.2 忘记密码页

#### 页面内容

- 导航栏：返回登录 + 标题"忘记密码"
- 步骤：① 输入账号 + 已绑定手机号 → ② 极验 + 短信验证码 → ③ 设置新密码 + 确认新密码 → ④ 完成

#### 校验规则

| 步骤 | 校验 | 错误提示 |
|---|---|---|
| 身份验证 | 账号与手机号必须匹配 | "账号或手机号错误" |
| 短信验证码 | 6 位数字，5 分钟有效，60 秒重发限制 | "验证码错误" / "验证码已失效" |
| 新密码 | 6-20 位字母+数字，不可与原密码相同 | "新密码不能与原密码相同" |

#### 交互行为

- 重置成功 → 自动跳回登录页，需用新密码重新登录

---

## 页面跳转关系

```
登录页 ──→ 工作台
   └──→ 忘记密码页 → 登录页
```
