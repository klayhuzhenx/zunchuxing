# 尊出行 · 运营端工作台模块 UI 设计说明

> PC Web 后台，登录后默认首页。不承载业务操作，所有操作通过卡片入口跳转。

---

## 页面清单

> 本模块共 **1 张页面**：工作台。

### 2.1 工作台

#### 页面内容（自上而下四个区域）

**顶部欢迎条**
- 头像 + 昵称 + 角色标签 + 当前时间

**核心数据卡片区（4-5 张卡片）**

| 卡片 | 数据 | 角色可见 |
|---|---|---|
| 今日订单 | 总订单量 / 较昨日对比 | 全部 |
| 今日交易额 | 实付总金额 / 较昨日对比 | 全部 |
| 待派车 | 当前待派车订单数 | 运营管理员、超级管理员 |
| 待补款 | 当前待补款订单数 | 财务管理员、运营管理员 |
| 在线司机 | 当前在线司机数 / 总司机数 | 运营管理员、超级管理员 |

**待办事项区（按优先级排列）**

| 事项类型 | 展示内容 | 跳转目标 |
|---|---|---|
| 待派车订单 | 数量 + 最近出发时间 | 订单管理 → 待派车 |
| 调度临近超时 | 距出发 ≤ 2.5h 未派车订单数 | 订单管理（高亮筛选） |
| 待审核证件 | 司机/车辆证件待审核数 | 司机/车辆管理 |
| 待审核企业入驻 | 待审核企业申请数 | 企业客户管理 → 待审核 |
| 退款待执行 | 已确认取消退款未完成数 | 财务管理 → 退款 |
| 额度告急企业 | 剩余额度低于阈值企业数 | 企业客户管理（额度筛选） |

**数据趋势区**
- 近 7 天订单量与交易额双折线图（横轴日期，左纵轴订单量，右纵轴交易额）
- 顶部切换按钮：近 7 天 / 近 30 天

#### 交互行为

- 点击数据卡片 → 跳转对应模块筛选视图
- 点击待办事项 → 跳转对应列表并自动应用筛选
- 数据 60 秒自动刷新，右上角"刷新"按钮可手动刷新
- 空状态（无待办）→ 显示"今日无待办事项"
- 数字对比：绿色 ↑ / 红色 ↓

#### 角色可见性

| 角色 | 可见区域 |
|---|---|
| 超级管理员 | 全部数据 |
| 运营管理员 | 订单、派车、司机、企业相关 |
| 财务管理员 | 交易额、退款、对账相关；不展示派车/司机 |
| 客服管理员 | 投诉、工单相关；不展示财务 |
