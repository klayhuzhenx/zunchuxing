# 尊出行 · 包车出行模块 UI 设计说明

---

## 页面清单

> 本模块共 **5 张页面**：起止点+车型、套餐弹窗、订单确认、费用明细、下单成功。

### 4.1 起止点 + 车型

#### 页面内容

- 导航栏：返回按钮 + 标题"包车出行"
- 出发地输入框，点击进入地址选择页（最近记录 + 搜索联想）
- 目的地输入框，同上
- 出发地与目的地交换按钮
- 车型选择区：横向滑动切换车型卡片
  - 增程星辉尊享版
  - 增程星辉行政版（待定）
  - 增程星耀行政版（待定）
- "请选择套餐"按钮（起止点未填时不可点击）

#### 交互行为

- 点击出发地/目的地 → 进入地址选择页 → 选中地址返回
- 点击交换按钮 → 交换出发地和目的地
- 横向滑动 → 切换车型
- 起止点填完后"请选择套餐"变为可点击 → 点击弹出套餐弹窗

---

### 4.2 套餐弹窗（底部弹窗）

#### 弹窗内容

- 弹窗标题：当前选中的车型名称（如"增程星辉尊享版"）
- 套餐卡片区：六张套餐卡片

  | 套餐 / 时长 | 半日租（4h/50km） | 日租（8h/100km） |
  |---|---|---|
  | 尊享基础 | ¥988 | ¥1,888 |
  | 尊荣高级 | ¥1,188 | ¥2,088 |
  | 尊御顶级 | ¥1,588 | ¥2,688 |

- 套餐权益标签区（随选中套餐切换）：

  | 套餐 | 权益标签 |
  |---|---|
  | 尊享基础 | 免费饮用水、车载 WiFi、纸巾湿巾 |
  | 尊荣高级 | 免费饮用水、车载 WiFi、纸巾湿巾、精致茶点、报刊杂志 |
  | 尊御顶级 | 免费饮用水、车载 WiFi、纸巾湿巾、精致茶点、报刊杂志、香槟礼遇、专属管家 |

- 包车天数：步进器（默认 1 天，范围 1-30 天）
- 起始日期选择器（默认当天，可选未来 30 天，需提前 ≥ 2 小时）
- 费用预览：套餐单价 × 天数 = 预估金额
- 提示文字："超时费 ¥100/小时，超公里费 ¥5/公里，按实际结算"
- "确认，查看订单"按钮

#### 交互行为

- 点击套餐卡片 → 选中套餐，权益标签同步更新
- 点击 +/- → 调整天数
- 点击起始日期 → 调起日期选择器
- 点击"确认，查看订单" → 关闭弹窗，跳转订单确认页

---

### 4.3 订单确认

#### 页面内容

- 导航栏：返回按钮 + 标题"确认订单"
- 行程路线信息：出发地地址、目的地地址
- 车型套餐信息：车型名称 + 套餐名称 + 租用类型（半日/日租） + 单价 × 天数
- 行程日期：起始日期至结束日期，共 N 天
- 乘车人：默认"本人"展示姓名+脱敏手机号，支持"切换"选择他人
  - 选择他人时展开：乘车人姓名输入框 + 乘车人手机号输入框
- 支付方式选择：
  - 个人支付（微信）
  - 个人支付（支付宝）
  - 企业支付（企业名称）（仅企业身份可见，企业身份默认选中）
- 备注输入框（选填）
- 协议文字："下单即表示同意《包车出行规则》"
- 底部固定栏：预估总金额 + "明细"入口 + "提交订单"按钮

#### 交互行为

- 点击乘车人"切换" → 弹出选择器，可选"本人"或填写"他人信息"
- 点击"明细" → 跳转费用明细页
- 选择支付方式 → 单选切换
- 企业支付额度不足 → Toast "企业额度不足，请联系管理员"
- 点击"提交订单" → 调起支付 → 支付成功跳转下单成功页
- 包车出行不支持尊享权益抵扣

---

### 4.4 费用明细

#### 页面内容

- 导航栏：返回按钮 + 标题"费用明细"
- 计费清单：
  - 套餐价（单价 × 天数）
  - 夜间附加（0:00-5:00 出发，套餐总价 ×20%，封顶 ¥1000）
  - 远调费：待定
  - 超时费（预估 ¥0，¥100/小时，按实际结算）
  - 超公里费（预估 ¥0，¥5/公里，按实际结算）
- 预估合计
- "查看计价规则"链接
- 底部提示："超时/超公里费按实际行程结算。套餐未用完时长/里程不退款。"

#### 交互行为

- 点击"查看计价规则" → 跳转 H5 页面

---

### 4.5 下单成功

#### 页面内容

- 成功图标
- 标题"下单成功"
- 副标题"正在为您匹配司机..."
- 订单摘要：订单号、车型套餐、日期范围、行程（出发地 → 目的地）
- "查看订单详情"按钮
- "返回首页"按钮

#### 交互行为

- 点击"查看订单详情" → 跳转订单详情页
- 点击"返回首页" → 跳转首页

---

## 页面跳转关系

```
首页 → 起止点+车型 → 套餐弹窗 → 订单确认 → 下单成功
                                  │
                                  └──→ 费用明细
```
