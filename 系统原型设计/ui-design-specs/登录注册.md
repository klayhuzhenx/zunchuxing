# 尊出行 · 登录注册模块 UI 设计说明

---

## 页面清单

> 本模块共 **6 张页面**：登录页、注册页、忘记密码页、微信授权登录（登录页内嵌按钮流程）、企业入驻页、企业入驻状态页。

### 2.1 登录页

#### 页面内容
- 品牌标识：品牌徽标（ZUNJIE·尊界）、主标题"尊出行"、副标题"高端出行服务平台"
- 手机号输入框
- 密码输入框（支持明文/密文切换）
- 协议勾选："我已阅读并同意《用户服务协议》和《隐私政策》"，协议名可点击跳转 H5
- "登录"按钮
- "微信快捷登录"按钮
- 底部入口："还没有账号？去注册"、"忘记密码？"、"企业入驻"

#### 交互行为
- 点击"登录" → 校验通过跳转首页；未注册跳转注册页，手机号自动带入；连续 5 次密码错误锁定 15 分钟
- 点击"去注册" → 注册页
- 点击"忘记密码" → 忘记密码页
- 点击"企业入驻" → 企业入驻页
- 点击"微信快捷登录" → 弹出微信授权弹窗
- 未勾选协议点击登录 → 弹出协议确认弹窗

#### 协议弹窗内容
- 标题："请阅读并同意以下协议"
- 说明文字："在使用尊出行服务前，请仔细阅读并同意："
- 协议列表：《用户服务协议》《隐私政策》（可点击跳转 H5）
- 按钮："不同意"、"同意并继续"

#### 协议弹窗交互
- 点击"同意并继续" → 自动勾选协议并执行登录
- 点击"不同意" → 关闭弹窗，留在登录页
- 点击遮罩区域 → 关闭弹窗

---

### 2.2 注册页

#### 页面内容
- 导航栏：返回按钮 + 标题"注册账号"
- 手机号输入框（从登录页跳转时自动带入）
- 验证码输入框 + "获取验证码"按钮
- 密码输入框（支持明文/密文切换）
- 协议勾选："我已阅读并同意《用户服务协议》和《隐私政策》"，协议名可点击
- "注册"按钮
- 底部入口："已有账号？去登录"

#### 交互行为
- 点击"获取验证码" → 弹出极验行为验证（滑块/点选） → 通过后发送短信 → 按钮进入 60s 倒计时
- 点击"注册" → 注册成功 → 获取微信用户信息授权 → 跳转首页
- 点击"去登录" → 登录页
- 手机号已注册 → 输入框下方提示"该手机号已注册" + "去登录"链接

---

### 2.3 忘记密码页

#### 页面内容
- 导航栏：返回按钮 + 标题"重置密码"
- 手机号输入框
- 验证码输入框 + "获取验证码"按钮
- 新密码输入框
- "提交"按钮

#### 交互行为
- 点击"获取验证码" → 同注册页验证码逻辑
- 提交成功 → Toast 提示 → 跳转登录页，手机号自动填入

---

### 2.4 微信授权登录

#### 触发位置
登录页"微信快捷登录"按钮

#### 弹窗内容
微信原生授权弹窗，内容为：
- 小程序名称及图标
- 授权说明："申请获取你的手机号码"
- 当前微信绑定的手机号（脱敏显示，如 138****8888）
- 按钮："拒绝"、"允许"

#### 交互行为
- 点击"允许" → 已绑定直接登录跳转首页；未绑定跳转手机号绑定页
- 点击"拒绝" → 关闭弹窗，留在登录页

---

### 2.5 企业入驻页

#### 页面内容
- 导航栏：返回按钮 + 标题"企业入驻"
- 企业名称输入框
- 统一社会信用代码输入框
- 联系人姓名输入框
- 联系人手机号输入框
- 验证码输入框 + "获取验证码"按钮
- 协议勾选："我已阅读并同意《企业入驻协议》"，协议名可点击
- "提交申请"按钮

#### 交互行为
- 提交成功 → 跳转入驻状态页
- 已有审核中/已通过的申请 → 直接跳转入驻状态页

---

### 2.6 企业入驻状态页

#### 页面内容
- 导航栏：返回按钮 + 标题"入驻状态"
- 状态展示区：
  - 审核中：时钟图标 + "审核中" + 预计完成时间说明
  - 已通过：对号图标 + "审核通过" + 成功说明
  - 已驳回：叉号图标 + "审核未通过" + 驳回原因（驳回态显示）
- 驳回态："修改重提"按钮
- "返回登录"按钮

#### 交互行为
- 点击"修改重提" → 回到入驻页，可修改信息重新提交
- 点击"返回登录" → 跳转登录页

---

## 页面跳转关系

```
登录页 ──→ 注册页
  ├──→ 微信授权弹窗（内嵌流程）→ 首页 或 手机号绑定页
  ├──→ 忘记密码页 → 登录页
  ├──→ 企业入驻页 → 入驻状态页
  └──→ 手机号绑定页（微信授权后未绑定手机号时跳入）→ 首页
```
