# 尊出行 · 行程管理模块 UI 设计说明

---

## 页面清单

> 本模块共 **2 张页面**：我的行程（订单列表）、订单详情。含取消订单弹窗和评价弹窗。

### 6.1 我的行程（订单列表）

#### 页面内容

- 标题栏：标题"我的行程"
- 搜索框："搜索订单"
- 状态 Tab：全部 | 待出发 | 行程中 | 已完成 | 已取消
- 订单卡片列表，每张卡片展示：
  - 出行场景标签（预约用车/包车出行/租车出行）
  - 行程摘要：
    - 预约用车：出发地 → 目的地
    - 包车出行：出发地 → 目的地 + 日期范围 + 天数
    - 租车出行：取车地址 + 还车地址 + 日期范围 + 天数
  - 状态标签（待支付/待调度/待出发/行程中/已完成/已取消）
  - 已取消订单显示取消原因 + "重新下单"按钮
- 空状态："暂无行程记录"

#### 状态 Tab 筛选规则

| Tab | 包含状态 |
|---|---|
| 全部 | 所有订单 |
| 待出发 | 待支付、待调度、已派车、待取车 |
| 行程中 | 行程中 |
| 已完成 | 已完成 |
| 已取消 | 已取消（含支付超时/调度超时/主动取消） |

#### 交互行为

- 点击 Tab → 按状态筛选订单列表
- 输入搜索框 → 模糊搜索订单
- 点击订单卡片 → 跳转订单详情页
- 点击"重新下单" → 跳转对应出行场景下单页，自动填入原订单信息

---

### 6.2 订单详情

#### 页面内容

- 导航栏：返回按钮 + 标题"订单详情"
- 订单状态 + 订单号
- 行程信息（按出行场景展示）：
  - 预约用车：出发地、目的地、途经点、出发时间
  - 包车出行：出发地、目的地、日期范围、共 N 天
  - 租车出行：取车地址、还车地址、日期范围
- 车型套餐信息：
  - 预约用车：车型名称
  - 包车出行：车型名称 + 套餐名称 + 日租单价 × 天数
  - 租车出行：车型名称 + 日租价 × 天数
- 包车订单额外展示：每日安排（按天平铺展示日期 + 时间 + 司机 + 车辆 + 当日状态）
- 租车订单额外展示：送取车任务状态（送车任务/取车任务各展示状态）
- 费用明细
- 底部操作区（按订单状态展示不同按钮）：
  - 待出发/待取车：取消订单
  - 已完成：评价行程
  - 已取消：重新下单

#### 交互行为

- 点击"取消订单" → 弹出取消确认弹窗 → 展示违约金 → 确认取消
- 点击"评价行程" → 弹出评价弹窗（司机 1-5 星、车辆 1-5 星、服务 1-5 星、文字评价选填 ≤200 字）
- 点击"重新下单" → 跳转对应出行场景下单页，自动填入原订单信息

#### 取消规则

| 场景 | 免费取消 | 扣 25% | 扣 50% | 扣 100% |
|---|---|---|---|---|
| 预约用车 | — | — | — | — |
| 包车出行 | 下单 20 分钟内或提前 ≥4h | — | 距出发 2-4h | 距出发 ＜2h |
| 租车出行 | 下单 20 分钟内或提前 ≥4h | — | 距出发 2-4h | 距出发 ＜2h |

> 包车已开始当日不可取消。取消后退款按支付方式原路返回。

## 页面跳转关系

```
我的 → 我的行程 → 订单详情
                     ├──→ 取消订单（弹窗）
                     ├──→ 评价行程（弹窗）
                     └──→ 重新下单 → 对应出行场景下单页
```
