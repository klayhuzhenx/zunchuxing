# 尊出行 · 预约用车模块 UI 设计说明

---

## 页面清单

> 本模块共 **4 张页面**：填写行程、确认订单、费用明细、下单成功。

### 3.1 填写行程

#### 页面内容

- 导航栏：返回按钮 + 标题"预约用车"
- 出发地输入框（带绿色圆点标识），支持文字搜索、地图选点、常用地址、历史地址
- 目的地输入框（带红色圆点标识），同上
- 出发地与目的地交换按钮
- "添加途经地点"入口（最多 10 个），途经点以灰色圆点+序号展示，支持拖拽排序和删除
- 出发日期选择器（默认当天，可选未来 30 天内）
- 出发时间选择器（5 分钟一档；选择当天时需 ≥ 当前时间 + 2 小时）
- 夜间时段（0:00-5:00）提示："该时段为夜间服务时段，里程费上浮 20%"
- 地图路线预览区域（选定出发地/目的地后自动描画路线）
- "查看路线详情"折叠面板（展示逐段距离和预计时间，默认收起）
- 常用地址标签：家、公司
- "确认路线，查看预估"按钮

#### 交互行为

- 输入地址 → 支持文字搜索联想、地图拖动选点、点击常用地址/历史地址
- 点击交换按钮 → 交换出发地和目的地
- 点击"+ 添加途经地点" → 展开途经点输入框
- 点击日期/时间 → 调起选择器
- 点击"查看路线详情" → 展开/收起路线详情
- 出发时间不满足提前 2 小时 → Toast "预约用车需提前至少 2 小时"
- 出发地/目的地不在服务区 → 提示"不在当前服务区域，暂不支持下单"
- 点击"确认路线" → 跳转确认订单页

---

### 3.2 确认订单

#### 页面内容

- 导航栏：返回按钮 + 标题"确认订单"
- 行程路线信息：出发地地址、目的地地址、途经点（如有）、预计距离和时间、出发日期时间
- 车型展示：尊界 S800 + "旗舰豪华轿车"
- 乘车人：默认"本人"展示姓名+脱敏手机号，支持"切换"选择他人
  - 选择他人时展开：乘车人姓名输入框 + 乘车人手机号输入框
- 尊享权益抵扣条（仅尊界车主且剩余次数 > 0 时显示）：
  - 显示"尊享权益 · 本次可抵扣"，剩余次数，"使用权益"按钮
  - 使用后费用变为 ¥0.00
- 支付方式选择：
  - 个人支付（微信）
  - 个人支付（支付宝）
  - 企业支付（企业名称）（仅企业身份可见，企业身份默认选中）
- 备注输入框（选填，提示文字"如有特殊需求可在此备注，如行李搬运等"）
- 协议文字："下单即表示同意《预约用车规则》"
- 底部固定栏：预估总金额 + "明细"入口 + "提交订单"按钮

#### 交互行为

- 点击乘车人"切换" → 弹出选择器，可选"本人"或填写"他人信息"
- 点击"使用权益" → 费用变为 ¥0.00，支付方式区域隐藏
- 点击"明细" → 跳转费用明细页
- 选择支付方式 → 单选切换
- 企业支付额度不足 → Toast "企业额度不足，请联系管理员"
- 点击"提交订单"：
  - 个人支付 → 调起支付（微信/支付宝）
  - 企业支付 → 直接扣减额度
  - 尊享权益 → 扣减 1 次权益
- 下单成功 → 跳转下单成功页

---

### 3.3 费用明细

#### 页面内容

- 导航栏：返回按钮 + 标题"费用明细"
- 计费清单：
  - 基础里程（≤20km）：¥300.00
  - 超里程（每满 5km +30 元）：按实际展示
  - 夜间附加（0:00-5:00，×20%，封顶 ¥1000）
  - 远调费：待定
  - 等候费（预估）：¥0.00（免费 15 分钟，超时每 5 分钟 +20 元，封顶 ¥2000）
- 预估合计
- "查看计价规则"链接（跳转运营后台 H5 页面）
- 底部提示："预估费用不等于最终费用：实际以行程结束里程为准。等候费按实际超时计算。"

#### 交互行为

- 点击"查看计价规则" → 跳转 H5 页面

---

### 3.4 下单成功

#### 页面内容

- 成功图标（绿色对号）
- 标题"下单成功"
- 副标题"正在为您匹配司机..."
- 订单摘要：订单号、出发时间、行程（出发地 → 目的地）
- "查看订单详情"按钮
- "返回首页"按钮

#### 交互行为

- 点击"查看订单详情" → 跳转订单详情页
- 点击"返回首页" → 跳转首页

---

## 页面跳转关系

```
首页 → 填写行程 → 确认订单 → 下单成功
                    │
                    └──→ 费用明细
```
