# 尊出行 · 我的模块 UI 设计说明

---

## 页面清单

> 本模块共 **2 张页面**：我的（主页面）、车主认证页。

### 我的（主页面）

#### 页面内容

- 用户信息区：头像、姓名、手机号（脱敏显示）
- 企业服务区（仅企业身份可见）：企业管理入口
- 常用功能区：
  - 我的行程
  - 电子发票
  - 常用地址
  - 消息中心（带未读角标）
- 车主认证入口（仅未认证的尊界车主可见）：展示"车主认证"入口，引导前往认证
- 设置区：客服中心、计价规则、用户服务协议、隐私政策、退出登录
- 底部 Tab 栏：服务 | 我的（当前页，选中态）

#### 交互行为

- 点击"我的行程" → 跳转订单列表页
- 点击"电子发票" → 跳转电子发票页
- 点击"常用地址" → 跳转常用地址管理页
- 点击"消息中心" → 跳转消息中心页
- 点击"企业管理" → 跳转企业管理页
- 点击"车主认证" → 跳转车主认证页
- 点击"客服中心" → 跳转客服页面
- 点击"计价规则" → 跳转 H5 规则页面
- 点击"用户服务协议" → 跳转 H5 协议页面
- 点击"隐私政策" → 跳转 H5 协议页面
- 点击"退出登录" → 弹出确认弹窗 → 确认后清除登录态 → 跳转登录页
- 点击"服务" Tab → 切换到首页

---

### 车主认证页

#### 页面内容

- 导航栏：返回按钮 + 标题"车主认证"
- 说明文字：认证说明
- 姓名输入框
- 身份证号输入框（18 位，末位 X 自动转大写）
- 手机号输入框（默认填充当前登录手机号，可修改）
- "提交认证"按钮

#### 交互行为

- 点击"提交认证" → 调用华为车主数据接口核验
  - 成功 → Toast "认证成功，尊享权益已生效" → 返回上一页
  - 失败 → Toast 具体错误原因（信息不匹配等）

---

## 页面跳转关系

```
首页（我的 Tab） → 我的（主页面）
                    ├──→ 我的行程（待补充）
                    ├──→ 电子发票（待补充）
                    ├──→ 常用地址（待补充）
                    ├──→ 消息中心（待补充）
                    ├──→ 企业管理（待补充）
                    ├──→ 车主认证页
                    ├──→ 各类 H5 页面（协议、规则、客服）
                    └──→ 退出登录 → 登录页
```
