# 尊出行 · 消息中心模块 UI 设计说明

---

## 页面清单

> 本模块共 **1 张页面**：消息列表。不设独立的消息详情页，点击消息直接跳转对应订单详情。

### 9.1 消息列表

#### 页面内容

- 导航栏：返回按钮 + 标题"消息中心" + 右上角"全部已读"按钮
- 消息卡片列表，每张展示：未读红点标识、标题、相对时间、出行场景标签、摘要（单行截断）
- 支持左滑删除
- 空状态："暂无消息"

#### 消息类型与触发场景

| 类型 | 触发时机 |
|---|---|
| 派车成功 | 后台分配司机和车辆后 |
| 行程开始 | 到达约定出发时间 |
| 行程结束 | 司机确认行程完成 |
| 取车提醒 | 租车订单取车时间前 1 小时 |
| 还车提醒 | 租车订单还车时间前 1 小时 |
| 订单取消 | 支付超时 / 调度超时自动取消 |

#### 交互行为

- 点击消息 → 标记已读 → 直接跳转对应订单详情
- 点击"全部已读" → 一键标记所有消息已读
- 左滑消息 → 展示删除按钮 → 确认删除
- 未读角标 → "我的"页面入口展示未读数，超 99 显示"99+"
