# 尊出行 · 服务页面模块 UI 设计说明

---

## 页面清单

> 本模块共 **1 张页面**：首页（服务 Tab）。

### 首页（服务 Tab）

#### 页面内容（从上到下）

**标题栏**
- 标题"尊出行"

**Banner 轮播区**
- 运营后台可配置的轮播图
- 暂无配置时此区域隐藏，下方内容自动上移

**公商务出行入口**
- 预约用车（副标题：单程点对点）
- 机场用车（副标题：接送机服务）
- 包车出行（副标题：4h / 8h 套餐）
- 租车出行（副标题：尊界车主专享）

**车主权益区**

未认证车主：
- 权益标题"尊享权益"
- 权益说明：每年免费出行次数、四大场景通用、专属车辆优先调度
- "前往认证 →"按钮

已认证车主：
- 权益标题"尊享权益"
- 剩余次数 + "剩余次数"标签
- 累计使用次数
- 权益标签卡片，每张包含标题和简短说明，如：
  - 年度 3 次免费 / 自然年自动重置
  - 四大场景通用 / 预约机场包车
  - 仅限本人使用 / 不可转让他人

**底部 Tab 栏**
- 服务（当前页，选中态）
- 我的

#### 交互行为

- 点击"预约用车" → 进入预约用车下单流程
- 点击"机场用车" → 进入机场用车下单流程
- 点击"包车出行" → 进入包车出行下单流程
- 点击"租车出行"（尊界车主） → 进入租车出行下单流程
- 点击"租车出行"（非车主） → Toast 提示"租车服务仅限尊界车主使用"
- 点击"前往认证 →" → 进入车主认证页
- 点击权益标签卡片 → 跳转权益详情页
- 点击"我的" Tab → 切换到我的页面
