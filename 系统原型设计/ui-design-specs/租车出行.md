# 尊出行 · 租车出行模块 UI 设计说明

---

## 页面清单

> 本模块共 **5 张页面**：取还车位置、车型+天数日期、订单确认、费用明细、下单成功。

### 5.1 取还车位置

#### 页面内容

- 导航栏：返回按钮 + 标题"租车出行"
- 取车位置输入框，点击进入地址选择页（最近记录 + 搜索联想）
- 还车位置输入框，同上
- "下一步，选择车型"按钮（取还车位置未填时不可点击）

#### 交互行为

- 点击取车位置/还车位置 → 进入地址选择页 → 选中地址返回
- 取还车位置填完后"下一步"变为可点击 → 点击跳转车型+天数日期页

---

### 5.2 车型 + 天数日期

#### 页面内容

- 导航栏：返回按钮 + 标题"选择车型与天数"
- 车型选择区：横向滑动切换车型卡片，每张卡片含车型名称 + 日租价 + 日里程

  | 车型 | 日租价格 | 日里程 | 超里程 |
  |---|---|---|---|
  | 增程星辉尊享版 | ¥1,500 | 含 200km | ¥10/公里 |
  | 增程星辉行政版 | ¥1,800 | 含 200km | ¥10/公里 |
  | 增程星耀行政版 | ¥2,200 | 含 200km | ¥10/公里 |

- 选中车型的名称和日租价展示在卡片下方
- 租用天数：步进器（默认 1 天，范围 1-30 天）
- 起始日期选择器（默认当天，可选未来 30 天）
- 费用预览：日租价 × 天数 = 预估金额
- 提示文字："超里程 ¥10/公里，按实际结算"
- "确认，查看订单"按钮

#### 交互行为

- 横向滑动 → 切换车型，下方展示信息同步更新
- 点击 +/- → 调整天数
- 点击起始日期 → 调起日期选择器
- 点击"确认，查看订单" → 跳转订单确认页

---

### 5.3 订单确认

#### 页面内容

- 导航栏：返回按钮 + 标题"确认订单"
- 取还车信息：取车地址、还车地址
- 车型信息：车型名称 + 日租价 × 天数
- 租期：起始日期至结束日期，共 N 天
- 驾驶人：默认"本人"展示姓名+脱敏手机号，支持"切换"选择他人
  - 选择他人时展开：驾驶人姓名输入框 + 驾驶人手机号输入框
- 驾驶证上传区：点击上传按钮调起相册或拍照，支持 JPG/PNG，≤10MB；上传后展示缩略图预览，可删除重传
- 尊享权益抵扣条（仅尊界车主且剩余次数 > 0 时显示）：
  - 显示"尊享权益 · 本次可抵扣"，剩余次数，"使用权益"按钮
  - 使用后费用变为 ¥0.00
- 支付方式选择：
  - 个人支付（微信）
  - 个人支付（支付宝）
  - 企业支付（企业名称）（仅企业身份可见，企业身份默认选中）
- 备注输入框（选填）
- 协议文字："下单即表示同意《租车出行规则》"
- 底部固定栏：预估总金额 + "明细"入口 + "提交订单"按钮

#### 交互行为

- 点击驾驶人"切换" → 弹出选择器，可选"本人"或填写"他人信息"
- 点击驾驶证上传区 → 调起相册或拍照 → 上传后展示缩略图 → 可点击删除
- 点击"使用权益" → 费用变为 ¥0.00，支付方式区域隐藏
- 点击"明细" → 跳转费用明细页
- 选择支付方式 → 单选切换
- 企业支付额度不足 → Toast "企业额度不足，请联系管理员"
- 点击"提交订单"：
  - 个人支付 → 调起支付（微信/支付宝）
  - 企业支付 → 直接扣减额度
  - 尊享权益 → 扣减 1 次权益
- 下单成功 → 跳转下单成功页

---

### 5.4 费用明细

#### 页面内容

- 导航栏：返回按钮 + 标题"费用明细"
- 计费清单：
  - 日租价（单价 × 天数）
  - 超里程费（预估 ¥0，¥10/公里，还车后按实际结算，日里程含 200km）
  - 远调费：待定
- 预估合计
- "查看计价规则"链接
- 底部提示："无夜间附加费、无等候费。超里程按实际超出结算。"

#### 交互行为

- 点击"查看计价规则" → 跳转 H5 页面

---

### 5.5 下单成功

#### 页面内容

- 成功图标
- 标题"下单成功"
- 副标题"订单已生成，等待派车..."
- 订单摘要：订单号、车型、租期、取还车地址
- "查看订单详情"按钮
- "返回首页"按钮

#### 交互行为

- 点击"查看订单详情" → 跳转订单详情页
- 点击"返回首页" → 跳转首页

---

## 页面跳转关系

```
首页 → 取还车位置 → 车型+天数日期 → 订单确认 → 下单成功
                                        │
                                        └──→ 费用明细
```
